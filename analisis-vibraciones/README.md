# Análisis de señales de vibración - Dominio del tiempo

Este proyecto forma parte de un proceso de aprendizaje orientado al análisis de señales aplicado a mantenimiento predictivo en equipos rotativos, como bombas industriales.

En este ejercicio se analiza una señal de vibración en el dominio del tiempo, simulando el comportamiento tipico de una bomba en operación normal. El objetivo es aprender como se adquieren, visualizan e interpretan señales temporales como primer paso para la deteccón temprana de anomalias mecánicas.
